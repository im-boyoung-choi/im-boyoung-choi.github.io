데이터 타입과 타입 변환

JavaScript에서는 여러 가지 형태의 타입을 미리 정의해서 제공하고 있으며 이를 기본 타입이라고 한다. 기본 타입은 크게 원시 타입(primitive type)과 객체 타입(object type)으로 구분할 수 있다.

오늘은 데이터 타입 중 원시 타입에 대해 살펴볼 예정이다. 객체 타입은 추후 객체 파트에서 따로 다룰 예정이니 해당 아티클을 살펴보도록 하자.

hello, JavaScript!

처음 블로그를 만든 목적은 패스트캠퍼스 ‘프론트엔드 개발 올인원 패키지’ 강의 중 JavaScript 파트인 ‘벨로퍼트와 함께하는 모던 자바스크립트’를 수강하며 개발 공부를 하는 것을 기록하기 위함이었다.

벨로퍼트 님의 GitBook을 참고해서 ‘1장. 자바스크립트 입문’의 마지막 파트를 공부하던 중 객체를 너무 겉핥기로 공부했다는 것을 알게됐다. 추가로 객체에 대해 공부하다 보니 벨로퍼트 님의 강의 문서만 따라해서는 초보자가 JavsScript를 쉽게 이해하기는 어려울 수 있다는 생각이 들었고, 작성했던 11개의 게시글을 지우고 다시 처음부터 정리를 시작한다.

벨로퍼트 님의 강의 문서를 토대로 작성하려고 했던 처음 목표와는 달리 나는 좀 더 초보자를 위해 TCP School 문서를 토대로 개념을 정리해가려 한다. 벨로퍼트 님의 강의에서 다루는 내용이 빠지는 것은 아니니 2개의 문서를 함께 참고하길 바란다.

TCP School의 커리큘럼은 JavaScript를 꼼꼼하게 학습하기에는 좋지만, 최신 동향인 모던 JavaScript 문법에 대해 학습하기에는 조금 옛날 버전이라는 느낌이 있다. 하지만 모던 JavaScript에 대해 바로 학습하는 것보다는 옛날 버전이라도 JavaScript의 전반적인 내용을 학습한 다음에 모던 JavaScript를 훑어보는 것이 더 기본기를 탄탄히 할 수 있을 거 같아 TCP School 커리큘럼을 따라간다.

나는 TCP School 커리큘럼 학습이 끝나면 모던 JavaScript 튜토리얼을 살필 예정이지만, 모던 JavaScript를 중점으로 학습하고 싶은 사람들은 모던 JavaScript 튜토리얼 커리큘럼을 따라 학습하는 것을 추천한다.
