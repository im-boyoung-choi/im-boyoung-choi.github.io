데이터 타입과 타입 변환

JavaScript에서는 여러 가지 형태의 타입을 미리 정의해서 제공하고 있으며 이를 기본 타입이라고 한다. 기본 타입은 크게 원시 타입(primitive type)과 객체 타입(object type)으로 구분할 수 있다.

오늘은 데이터 타입 중 원시 타입에 대해 살펴볼 예정이다. 객체 타입은 추후 객체 파트에서 따로 다룰 예정이니 해당 아티클을 살펴보도록 하자.

1. 데이터 타입(data type)

데이터 타입(data type)은 프로그램에서 다룰 수 있는 값의 종류를 의미한다.

우리는 변수나 상수를 선언할 때 숫자가 아닌 다른 값들도 선언할 수도 있다. 숫자(number), 문자열(string), 참/거짓(boolean), 없음(null/undefined)을 원시 타입(primitive type)이라 하고 이를 제외한 대부분을 객체 타입(object type)이라고 한다.

숫자(number)

변수와 상수 파트 예제에서 봤던 것과 같이 숫자를 선언하는 것이다. 숫자 있는 그대로 값을 대입하면 된다.

참/거짓(boolean)

참은 true 거짓은 false로 값을 나타내는 것을 우리는 boolean이라고 한다.

없음(null/undefined)

JavaScript에서 ‘없음’을 의미하는 데이터 타입으로 약간의 의미 차이가 있다.

null은 객체 타입으로 값이 없음을 의미, undefined는 값/타입이 아직 정의되지 않았음을 의미한다. 즉, undefined는 존재하지 않는 값에 접근할 때 반환된다.

심볼(Symbol) 타입과 typeof 연산자

심볼 타입은 ECMAScript 6부터 새로 추가된 타입이다. 심볼은 유일하고 변경할 수 없는 타입으로, 객체의 프로퍼티를 위한 식별자로 사용할 수 있다. 익스플로러 브라우저를 지원하지 않는다.

typeof 연산자는 피연산자가 하나일 때, 피연산자의 타입을 반환하는 연산자이다.

2. 데이터 타입 변환

JavaScript는 타입 검사가 매우 우연한 언어다. 변수의 타입이 정해져 있지 않으며, 같은 변수에 다른 타입의 닶을 다시 대입할 수도 있다.

묵시적 타입 변환(implicit type conversion)

특정 타입의 값을 기대하는 곳에 다른 타입의 값이 오면 자동으로 타입을 변환시킨다. 즉, 문자열 타입이 오길 기대하는 곳에 숫자 타입이 오더라도 알아서 숫자를 문자열로 변환시키는 것이다.

10 + '텍스트'를 보면 텍스트를 연결해주는 덧셈 연산자 사용으로 인해 숫자로 넘겨받은 10을 문자열로 변환시켜 결과를 반환한다.

덧셈 연산자가 아닌 그 외의 연산자는 '10' * '10'처럼 문자열로 숫자를 넘겨줘도 숫자로 받아 처리하는 것을 확인할 수 있다.

10 - '텍스트'는 뺄셈 연산을 위해 문자열이 숫자로 변환되어야 하지만 해당 문자열은 숫자로 변환될 수 없는 문자열이기 때문에 타입을 변환하지 못했고 NaN 값을 반환했다.

NaN
NaN(Not a Number)은 잘못된 입력으로 인해 계산할 수 없음을 나타내는 기호이다.

명시적 타입 변환(explicit type conversion)

전역 함수를 이용해 명시적으로 타입을 변환할 수도 있다. 타입을 변환할 때 사용하는 전역 함수에 대해 살펴보자.

Number(object)
받은 인자를 숫자로 변환한다. 숫자만 포함한 문자열은 숫자 타입으로, 텍스트가 포함된 문자열은 NaN이 반환된다. 참/거짓(boolean)을 다룰 때는 true는 1, false는 0을 반환한다.

parseInt(string)
문자열을 인자로 받아 정수로 변환핸다. 문자열에 포함된 텍스트가 전부 숫자가 아니어도 숫자로 시작하는 경우에는 숫자가 끝나는 지점까지 변환한다. 숫자로 시작하지 않는 문자열은 숫자를 포함하더라도 NaN이 반환된다.

parseFloat(string)
문자열을 인자로 받아 실수로 변환한다. 문자열에 포함된 텍스트가 전부 숫자가 아니어도 숫자로 시작하는 경우에는 숫자가 끝나는 지점까지 변환한다. 숫자로 시작하지 않는 문자열은 숫자를 포함하더라도 NaN이 반환된다.

Date() 생성자를 이용해서 날짜 정보를 숫자로 변환

String(object)
받은 인자를 문자열로 변환한다.

숫자를 문자열로 변환하는 메소드

toString() : 변수의 값을 문자열로 반환한다.

toExponential() : 소수를 지수표기법(Exponnential Notation)을 사용하여 문자열로 반환, 소수점 몇 번째 자리까지 리턴할 것인지를 매개변수로 받는다.

toFixed() : 소수를 문자열로 반환, 소수점 몇 번째 자리에서 반올림할 것인지를 매개변수로 받는다.

toPrecision() : 소수를 문자열로 반환, 몇 번째 자리까지 리턴할 것인지를 매개변수로 받아 리턴한다. 해당 지점이 정수 지점이면 지수표기법으로 반환, 소수점 자리라면 반올림하여 반환한다.

Boolean(object)
받은 인자를 true 혹은 false로 반환한다.

false로 값을 반환하는 경우는 빈 문자열 타입을 받았을 때, 0 또는 NaN 숫자 타입을 받았을 때, null 객체 타입을 받았을 때, null/undefined 값을 받았을 때다. 이를 제외한 값에서는 true를 반환한다.
