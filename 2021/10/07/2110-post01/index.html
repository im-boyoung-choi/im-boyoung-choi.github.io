05. 함수

여기까지는 ‘수학의 정석’ 맨 앞 챕터와 같은 범위라고 생각한다. JavaScript를 공부하려고 마음먹은 누구나 if 문까지 살펴보지 않은 사람 없을 것이다.

사실 오늘 들어가는 함수도 어려운 것은 아니지만 본격 JavaScript 공부를 시작하는 느낌이다. 지금부터 함수에 대해 살펴보자.

1. 함수

함수란 특정 코드를 하나의 명령으로 실행할 수 있게 해주는 기능이다. 예를 들어 특정 값을 더한 값을 알고 싶을 때 우리는 아래와 같은 코드를 작성한다.

이 작업을 함수로 변환해 보자.

함수를 만들 때에는 function 키워드를 사용하며, 함수에서 어떤 값을 받아올지 정해주는 역할을 하고 이를 파라미터(매개변수)라고 한다.

함수 내부에서 사용되는 return 키워드는 함수의 결과물을 지정하는 역할을 한다. return 키워드를 사용하면 함수가 끝나기 때문에 아래와 같은 코드를 작성하면 return 아래의 코드는 호출되지 않는다.

연산자를 배우면서 문자열을 조합할 때 + 연산자를 사용해도 된다는 것을 알게 됐다. 하지만 더 쉬운 방법이 있다. ES6 의 템플릿 리터럴(Template Literal)이라는 문법을 활용하는 것이다.

ES6 란?

ES6 란 ECMAScript6 의 약어로 2015년에 도입된 자바스크립트 버전을 의미한다. (2015년에 도입되어 ES2015 라고 불리기도 함) 2015년 이후에 계속해서 ES7(ES2016) ES8(ES2017) ES9(ES2018) ES10(ES2019) 등 새로운 버전이 나오고 있다.

브라우저 버전에 따라 지원되는 자바스크립트 버전이 다르지만 웹 개발을 할 때에는 Babel 이라는 도구를 사용하여 최신 버전의 자바스크립트가 구 버전의 브라우저에서도 실행되도록 할 수 있다. (정확히는 변환하는 작업을 거치는 것)

3. 화살표 함수
