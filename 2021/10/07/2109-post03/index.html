반복문

반복문(iteration sstatements)이란 똑같은 명령을 일정 횟수만큼 반복하여 수행하도록 제어하는 실행문이다.

프로그램이 처리하는 대부분의 코드는 반복적인 형태가 많기 때문에 가장 많이 사용되는 실행문 중 하나다. 반복문의 종류와 쓰임에 대해 알아보자.

1. for 문

for 문은 가장 기본적인 반복문이다. 특정 값에 변화를 주면서 우리가 정한 조건이 만족할 경우 계속해서 반복한다. 아래 코드를 살펴보자.

for 문의 변화 구문은 보통 i++를 통해 1 씩 값을 증가시켜 사용한다. 반대로 i--를 통해 1 씩 값을 감소시키는 형태도 가능하다. 아래 코드를 살펴보자.

2. for…of 문

for…of 문은 후에 배울 배열에 관한 반복문을 돌리기 위해 만들어진 반복문이다.

보통 배열을 반복할 때는 배열의 내장함수를 사용하기 때문에 실제로 사용할 일은 별로 없다고는 하나 가볍게 원리라도 알고 넘어가도록 하자.

3. for…in 문

for…in 문은 객체를 위한 반복문이다. 정확한 구문을 알아보기 전에 객체의 정보를 배열 형태로 받아오는 함수 몇 가지를 살펴보자.

대소문자를 구분하는 함수이며, 각 함수의 역할은 다음과 같다.

  • Object.entries : [[키, 값], [키, 값]] 형태의 배열로 반환
  • Object.keys : [키, 키, 키] 형태의 배열로 반환
  • Object.valus : [값, 값, 값] 형태의 배열로 반환

객체가 가지고 있는 값에 대하여 반복을 하고 싶다면 위 함수들을 사용해도 되고 for…in 문을 사용해도 된다. 아래 코드를 통해 for…in 문은 어떻게 사용하는 것인지 살펴보자.

4. while 문

while 문은 특정 조건이 참이라면 계속해서 반복하는 반복문이다.

for 문은 특정 숫자를 가지고 숫자의 값을 비교하고 증감하면서 반복이 된다면, while 문은 조건을 확인만 하면서 반복이 진행된다. 그래서 while 문에서는 조건문 내부에서 변화를 직접 줘야 한다. 아래 코드를 살펴보자.
