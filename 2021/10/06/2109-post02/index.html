04. 조건문

특정 조건이 만족했을 때 특정 코드를 실행하는 구문을 조건문이라 한다. 우리가 흔히 알고 있는 if 문부터, switch 문까지 이번 순서에서 살펴보자.

1. if 문

if 문은 ‘~ 한다면, ~ 해라’는 의미를 지닌 가장 기본적인 조건문이다. 바로 예시 코드부터 살펴보자.

a + 1 은 2 입니다.가 출력된다. 다음 코드도 살펴보자.

처음과 달리 아무것도 출력되지 않는다. 이렇듯 if 문을 사용하면 특정 조건이 만족 될 때만 특정 코드를 실행할 수 있다.

조건이 만족됐을 때 실행시킬 코드는 { } 를 통해 감싸져 있다. 이를 코드 블록이라고 한다.

만약 조건이 true가 된다면 지정한 코드가 실행, false가 된다면 실행되지 않는다.

이전에 ‘변수와 상수’에서 let과 const를 배울 때 같은 이름으로 2번을 선언할 수는 없다고 배웠는데, 이 코드 블록을 기준으로 다른 블록 범위에서는 같은 이름으로 선언하는 것이 가능하다. 아래 코드를 살펴보자.
